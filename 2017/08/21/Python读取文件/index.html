Python读取文件

2017-08-21

python中读取文件常用的三种方法：read(),readline(),readlines()。在做笔试题的时候被这个东西卡了半天，遂总结一下。

假设a.txt的内容如下所示：

This is line1.
This is line2.
This is line3.

#read()
语法如下：

read([size])

从文件当前位置起读取size个字节，若无参数size，则表示读取至文件结束为止，它范围为字符串对象。

f = open("a.txt")
lines = f.read()
print lines
print(type(lines))
f.close()

输出结果：

This is line1.
This is line2.
This is line3.
<type 'str'> #字符串类型

readline()

该方法每次读出一行内容，所以，读取时占用内存小，比较适合大文件，该方法返回一个字符串对象。

f = open("a.txt")
line = f.readline()
print(type(line))
while line:
    print line,
    line = f.readline()
f.close()

输出结果：

<type 'str'>
This is line1.
This is line2.
This is line3.

#readlines()
读取整个文件所有行，保存在一个列表(list)变量中，每行作为一个元素，但读取大文件会比较占内存。
f = open(“a.txt”)
lines = f.readlines()
print(type(lines))
for line in lines:
print line，
f.close()
输出结果：

<type 'list'>
This is line1.
This is line2.
This is line3.
