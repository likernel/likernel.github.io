Python中的range和xrange

介绍Python中的range和xrange以及它们之间的区别。

range

函数说明：range([start,] stop[, step])，根据start与stop指定的范围以及step设定的步长，生成一个序列。

xrange

函数说明：用法与range完全相同，所不同的是生成的不是一个数组，而是一个生成器。

xrange和range区别

这两个基本上都是在循环的时候用。

要生成很大的数字序列的时候，用xrange会比range性能优很多，因为不需要一上来就开辟一块很大的内存空间。尽量用xrange，除非你是要返回一个列表。

备注：在Python 3中，range()是像xrange()那样实现以至于一个专门的 xrange()函数都不再存在（在python 3中xrange()会抛出命名异常）。
